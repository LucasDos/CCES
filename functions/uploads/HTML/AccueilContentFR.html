Bienvenue

Le catalogue des cours est destiné aux étudiants d'échanges entrants.

Attention : ce catalogue répertorie les cours les plus suivis par les étudiants d’échanges ; il ne concerne pas l’ensemble de l’offre de formation de l’Université de Tours.

Vous pouvez consulter l’offre de formation complète en cliquant sur le lien suivant : https://www.univ-tours.fr/formations/offre-de-formation/

Pour les cours enseignés en français, le niveau de français recommandé pour une compréhension idéale de ces cours est le B2 du cadre européen de référence.

Les étudiants d'échanges peuvent également choisir quelques cours de français au CUEFEE (Centre universitaire de français pour étudiants étrangers) : un maximum de 12 crédits ECTS peut être validé.

Le catalogue permet de consulter les cours offerts par les facultés de l'Université de Tours, aux niveaux Licence et Master. La recherche est possible par niveau, semestre, discipline, langue d'enseignement, etc. Des mises à jour sont effectuées deux fois par an, avant le début de chaque semestre.
